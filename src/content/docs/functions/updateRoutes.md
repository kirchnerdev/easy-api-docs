Updates the existing routes added by the loader.
THIS WILL REMOVE ANY ROUTE ADDED THROUGH THE MAIN FILE!
## Parameters
| Name |       Description       | Enforced | Default Value |
|------|-------------------------|----------|---------------|
| Path | The path to the routes. | Yes      | None          |
## Example
```eats
$updateRoutes[path]
```
