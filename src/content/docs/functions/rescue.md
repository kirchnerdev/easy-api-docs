catch statement as blocks.
:::danger
**$rescue** is only usable inside **$xtry**.
:::
## Parameters
| Name |           Description            | Enforced | Default Value |
|------|----------------------------------|----------|---------------|
| Code | The code to execute when rescue. | Yes      | None          |
## Example
```eats
$rescue[code]
```
