Loads data to be available globally.
## Parameters
| Name  |          Description           | Enforced | Default Value |
|-------|--------------------------------|----------|---------------|
| Name  | Name of the data to be loaded. | Yes      | None          |
| Value | The value to be loaded.        | Yes      | None          |
## Example
```eats
$loadGlobalData[name;value]
```
